# Unit-Converter
A Qt5 application which allows the user to convert a large variety of Units

## Dependencies:

-   PyQt5
-   Classes folder (obviously)

that's it.

## To run:

Run the command:
```cmd
UnitConverter.py
```
Make sure you have the `Classes` folder as it has dependencies in it.
